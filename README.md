Este repositório tem como objetivo praticar **Code Review** e o uso do **GitHub**, simulando um fluxo de trabalho semelhante ao utilizado em ambientes profissionais de desenvolvimento.

---

## Objetivos

- Praticar abertura e revisão de Pull Requests  
- Simular fluxo de trabalho colaborativo  
- Aplicar boas práticas de versionamento  
- Melhorar a qualidade do código através de Code Review  

---

## Tecnologias e Ferramentas

- Git  
- GitHub  
- HTML
- CSS
- JavaSxript

---

## Fluxo de Trabalho

1. Criar uma branch a partir da `main`  
2. Desenvolver a funcionalidade ou correção  
3. Fazer commits seguindo boas práticas  
4. Abrir um Pull Request  
5. Realizar o Code Review  
6. Fazer merge após aprovação  

---

## Boas Práticas

- Commits pequenos e descritivos  
- Código limpo e organizado  
- Revisões construtivas  
- Uso adequado de branches  

---

## Objetivo Final

Simular o dia a dia de um time de desenvolvimento, adquirindo experiência prática com **GitHub** e **Code Review**.
